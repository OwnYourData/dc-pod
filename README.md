# Data Intermediary Storage Pod
Implementation of an open source storage pod used by a Data Intermediation Service Provider.

## Issues

Please report bugs and suggestions for new features using the [GitHub Issue-Tracker](https://github.com/OwnYourData/dc-pod/issues) and follow the [Contributor Guidelines](https://github.com/twbs/ratchet/blob/master/CONTRIBUTING.md).

If you want to contribute, please follow these steps:

1. Fork it!
2. Create a feature branch: `git checkout -b my-new-feature`
3. Commit changes: `git commit -am 'Add some feature'`
4. Push into branch: `git push origin my-new-feature`
5. Send a Pull Request
